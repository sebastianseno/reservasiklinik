# Reservasi Mobile Apps

Library included on this Apps :
- Hilt (for Dependecy Injection)
- Retrofit, OkHttp (for Networking)
- Coroutine (for asynchronous)
- Coil (for load image)
- jetpack (for navigation Controller)
